01 — Contexto

O que é o TabNet?

Sistema de tabulação de dados do DATASUS / Ministério da Saúde
1
Base de dados pública — concentra indicadores de saúde do Brasil: internações, procedimentos, óbitos, vacinas e mais.
2
Sem API disponível — os dados só podem ser acessados por meio de um formulário web interativo, sem endpoint para download direto.
3
Processo manual é inviável — selecionar mês a mês, coluna a coluna, para gerar séries históricas levaria horas.
4
Solução: Web Scraping — automatizar o navegador para reproduzir os cliques e capturar o CSV gerado pela página.
Foco deste projeto: SIH — Sistema de Informações Hospitalares, filtrando por procedimento e especialidade nos últimos N meses.
02 — Objetivo

O que o script entrega?

Baixar séries históricas do SIH para os últimos N meses, organizadas por todas as combinações de:

DimensãoOpções coletadasExemplo de dado
ColunaProcedimento realizado, EspecialidadeCirurgia cardíaca
ConteúdoQuantidade aprovada, Valor aprovado1.240 internações
PeríodoÚltimos N meses (padrão: 13)Jan/2025 … Jan/2026
Saída: arquivos .csv por combinação, separados por ;, com coluna ano indicando o período — prontos para Excel, Power BI ou Python.

07 — Próximos Passos

Playwright vs Selenium

Por que migrar para o Playwright?
CritérioSeleniumPlaywright
Espera por elementos Manual — exige WebDriverWait em cada ação Auto-wait nativo — espera automaticamente o elemento estar pronto
Troca de abas Detectar e trocar de janela manualmente Captura a nova página com page.wait_for_event('popup')
Modo headless Configuração extra no ChromeOptions Headless por padrão, sem configuração adicional
Velocidade Mais lento — protocolo WebDriver sobre HTTP Mais rápido — comunica diretamente via WebSocket (CDP)
API Verbosa — muitas linhas para ações simples Fluente — page.click(), page.fill(), page.content()
